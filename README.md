# ghostty config

Personal [Ghostty](https://ghostty.org) terminal configuration.

## Settings

|Key|Value|
|---|-----|
|Font|MesloLGS NF|
|Font Size|13.5|
|Background Opacity|0.75|
|Background Blur|40|

## Installation

```sh
git clone <repo-url> ~/.config/ghostty
```

Ghostty automatically reads from `~/.config/ghostty/config` on macOS.
